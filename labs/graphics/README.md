### Взаимодействие
Есть три режима:
- Клавиша L — основной (Light) режим, все источники света работают;
- Клавиша P — режим рисования полигонов. Для создания вершины нажать ЛКМ, для завершения рисования текущего полигона — нажать ПКМ;
- Клавиша T — режим установки новых источников света. Нажать в точке ЛКМ для установки.

Также есть следующие функции:
- Клавиша B — отображать ли границы полигонов;
- Клавиша D — удаляет последний установленный источник света;
- Клавиша C — удаляет последний созданный полигон;
